# 📡 ZenRadar — Decentralized E2EE Bookmark & Read-Later Hub

ZenRadar is a lightweight, zero-knowledge **Bookmark & Read-Later Manager** running serverless inside the browser (Pocket & Raindrop.io P2P alternative). It secures all saved web links, tags, and articles locally using WebCrypto API before syncing them across devices via `@akaoio/zen`.

Powered by **ZEN**—a zero-config decentralized peer-to-peer graph database.

## 🚀 Features

- 📡 **Bookmark Vault**: Save web links with URL, Title, Description, Tag pills, and Favorite flag.
- 📖 **Read / Unread State**: Mark articles as read or unread to keep track of your reading backlog.
- 🔗 **One-Click Launch & Copy**: Instantly open links in a new tab or copy URLs to clipboard.
- 🔐 **End-to-End Encrypted**: Zero-knowledge bookmarks private to your public key.
- ⚡ **Real-Time P2P Sync**: Live bookmark sync across desktop and mobile browsers.
